# Orshemtov Brew

## How do I install these formulae?

`brew install orshemtov/brew/<formula>`

Or `brew tap orshemtov/brew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "orshemtov/brew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
